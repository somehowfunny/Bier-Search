# Bier Search

## Project Overview
Bier Search is a web application designed to help users discover and explore various types of beers. Users can search for beers based on different criteria, view details about their favorite brews, and find local breweries.

## Features
- **Search Functionality:** Easily search for beers by name, type, or brewery.
- **Beer Details:** Get detailed information about each beer, including alcohol content, flavor profile, and more.
- **Brewery Listings:** Discover local breweries and their offerings.
- **User Reviews:** Leave and read reviews from other beer enthusiasts.

## Setup Instructions
1. **Clone the repository:**
   ```bash
   git clone https://github.com/somehowfunny/Bier-Search.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Bier-Search
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the application:**
   ```bash
   npm start
   ```

5. **Open your browser:** Go to `http://localhost:3000` to view the application.

Enjoy exploring the wonderful world of beers!
